#Project set up
Install typescript locally  
```npm i typescript --save-dev```

Initialize typescript project  
```tsc --init```

Adjust tsconfig.json  

Install tslint locally   
```npm i tslint --save-dev```

Init tslint  
```npx tslint --init```

Build project  
```tsc```
